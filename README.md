# 家政服务管理系统简介 python1077

## 项目概述


## 技术栈

- 后端：Python + Django
- 前端：Vue.js
- 数据库：MySQL

## 功能模块

系统主要分为以下功能模块：

1. 用户管理
2. 家政人员管理
3. 服务项目管理
4. 订单管理
5. 支付管理
6. 评价与投诉

## 系统角色

- 管理员：负责系统的维护与管理
- 家政人员：提供家政服务，管理个人资料和订单
- 用户：享受家政服务，发布订单和评价

## 运行环境

- 后端：建议使用 Python 3.6 或以上版本
- 前端：要求 Node.js 版本高于 14
- 数据库：MySQL 5.7 或以上版本

## 部署步骤

1. 安装 Python 和 Django
2. 安装 MySQL 数据库
3. 部署前端 Vue.js 项目
4. 配置后端与前端通信
